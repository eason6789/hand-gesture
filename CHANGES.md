### 4. 漩涡动画 + AI 并行调用

新的解读流程：
1. 打开结果弹窗，显示卡牌
2. **立即启动漩涡粒子动画**（Canvas 80粒子 + 能量环 + 符文名轨道旋转）
3. **同时发起 AI API 请求**（不再等动画结束）
4. AI 返回或 8 秒超时后，漩涡淡出，显示解读内容

旧流程是"仪式动画 → 显示结果 → 请求AI"，现在改为"显示结果 + 动画 + 请求AI 三者并行"。

漩涡动画特征：
- 中心脉冲光晕
- 4 层旋转能量环（虚线）
- 80 个螺旋粒子向外扩散
- 选中符文名绕中心轨道运行
- 外层辉光环呼吸效果

### 5. 手势识别计时优化

| 手势 | 所需时间 | 帧数 (@60fps) | 说明 |
|------|----------|---------------|------|
| 🤏 捏合选中 | 2 秒 | 120 帧 | 衰减从 -15/帧 降至 **-3/帧**，避免检测抖动导致计数器反复归零 |
| ✊ 握拳清空 | 5 秒 | 300 帧 | 同上，衰减从 -10/帧 降至 **-3/帧** |
| 🖐 张开手掌提交 | 3 秒 | 180 帧 | 无变化 |

**核心修复**：捏合检测中 `pinchDist < 0.30` 可能因手部微小抖动而短暂失效。旧代码每次失效就减去 15 帧（0.25 秒），导致实际等待时间远超 2 秒。新代码改为 -3 帧，容忍手部自然抖动。

### 6. AI 解读结构增强

新的 Prompt 结构包含：

1. **整体能量场感知** — 元素流转趋势 + 阴阳平衡判断
2. **顺序解读** — 每枚古符结合其在序列中的位格含义
3. **古符共鸣分析** — 卡牌间的呼应、冲突、递进关系
4. **命运箴言** — 综合所有符纹主题

每枚卡牌携带的上下文信息：
- 选中顺序（"第 N 枚"）
- 元素属性（木火土金水 + 描述）
- 阴阳极性
- 与前一符的能量关系

本地 fallback `generateReading()` 同步升级：
- 元素流转分析
- 阴阳平衡判断
- 位格含义描述（10 种位格各有专属解读）
- 卡牌间两两共鸣解析
- 按选中数量自适应箴言措辞
